# 🏛️ METAIMPERIYA — Социальная платформа нового поколения

![METAIMPERIYA](https://raw.githubusercontent.com/Metaimperiya/MyWebSite-MyEcosystem/main/assets/images/metaimperiya-icon.png)

**METAIMPERIYA** — современная социальная сеть с полноценным функционалом для общения, управления друзьями и создания сообществ.

## 🚀 Функционал

- ✅ Регистрация через Google или анонимно
- ✅ Лента постов с картинками и ссылками
- ✅ Комментарии с редактированием и удалением
- ✅ Друзья (запросы, принятие, удаление)
- ✅ Группы/комнаты с общим чатом
- ✅ Личные сообщения
- ✅ Музыкальный плеер с плейлистом
- ✅ Профиль с аватаркой
- ✅ Уведомления
- ✅ Админ-панель
- ✅ PWA (можно установить на телефон)
- ✅ Мобильный адаптив

## 🛠️ Установка

1. Клонируй репозиторий
2. Настрой Firebase (создай проект, включи Realtime Database, Storage, Auth)
3. Замени ключи в `.env` и `js/firebase.js`
4. Залей на Vercel или GitHub Pages
