📄 Инструкция

  • Каждая страница — это отдельная папка с index.html
  • Структура: /user/username/index.html или /group/groupname/index.html
  • Для создания страницы просто скопируй шаблон и отредактируй
  • Все изменения делаются вручную через GitHub
  • Firebase используется только для счётчиков и интерактива
